# TradingView Charting Library 배치 안내

이 폴더는 TradingView Advanced Charts / Charting Library 원본 파일을 두는 위치입니다.

TradingView Charting Library는 공식 접근 승인이 필요한 라이브러리이므로 이 저장소에 원본 파일을 포함하지 않습니다. TradingView에서 승인받은 계정으로 공식 저장소에 접근한 뒤, 배포 가능한 파일을 이 폴더에 복사하세요.

앱은 다음 파일을 순서대로 찾습니다.

1. `charting_library.standalone.js`
2. `charting_library.js`

파일이 없으면 화면에는 한국어 안내 문구가 표시됩니다. 임의 출처의 파일을 다운로드하거나 커밋하지 마세요.
